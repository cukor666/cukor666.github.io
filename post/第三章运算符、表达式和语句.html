3.1 运算符与表达式

Java提供了丰富的运算符，Java中绝大多数运算符和C语言相同，基本语句也和C语言类似。

3.1.1 算术运算符与算术表达式

  • 加：+
  • 减：-
  • 乘：*
  • 除：/

3.1.2 自增、自减运算符

  • 自增：++
  • 自减：–

3.1.3 算术混合运算的精度

精度从”低“到”高“排序的顺序为

byte->short->char->int->long->float->double

在Java中计算算术表达式的值时使用下列精度运算规则：

  1. 如果表达式中有双精度浮点数，则按双精度进行运算
  2. 如果表达式中的最高精度是单精度浮点数，则按单精度进行运算
  3. 如果表达式中的高精度是long型整数，则按long精度进行运算
  4. 如果表达式中的最高精度低于int型整数，则按int精度进行运算

Java允许不超过byte、short和char的取值范围的常量算术表达式的值赋给byte、short和char型变量。

3.1.4 关系运算符与关系表达式

关系运算符是二目运算符，用来比较两个值的关系。关系运算符的运算结果是boolean型，当运算符对应的关系成立时，运算结果是true，否则为false.

运算符优先级用法含义结合方向
>6a > b大于从左向右
<6a < b小于从左向右
>=6a >= b大于等于从左向右
<=6a <= b小于等于从左向右
==7a == b等于从左向右
!=7a != b不等于从左向右

3.1.5 逻辑运算符与逻辑运算符

运算符优先级用法含义结合方向
&&11a && b逻辑与从左向右
|12a |b
!2!a逻辑非从右向左

3.1.6 赋值运算符与赋值表达式

和C一样

int a = 4;
double b = 3.14;
char c = 'A';
double d = a + b;

3.1.7 位运算符

和C一样

  • 按位与：& 有假则假
  • 按位或：| 有真则真
  • 按位异或：^ 不同为真，相同为假

3.1.8 instanceof运算符

instanceof运算符是二目运算符，左面的操作元是一个对象，右面是一个类。==当左面的对象是右面的类或子类创建的对象时，该运算符运算的结果是true，否则是false.==

3.1.9 运算符综述

Java表达式就是用运算符连接起来的符合Java规则的式子。运算符的优先级决定了表达式中运算执行的先后顺序。和C一样。

优先级表如下：

优先级描述运算符
1分隔符[]、()、.、,、;
2对象的归类，自增，自减逻辑非instanceof、++、–、!
3算术乘除运算*、/、%
4算术加减运算+、-
5移位运算符>>、<<、>>>
6大小关系运算<、<=、>、>=
7相等关系运算==、!=
8按位与运算&
9按位异或运算^
10按位或运算|
11逻辑与运算&&
12逻辑或运算|
13三目运算符?:
14赋值运算符=

3.3 if条件分支语句

条件分支语句按语法格式可细分为3种形式，以下是这3种形式的详细讲解。

  • if 语句
  • if-else 语句
  • if-else if-else语句
  • switch开关语句

基本用法和C语言是一样的，唯一的区别就是Java中判断表达式的结果必须是Boolean类型，而C中的是非0为真。

if (判断表达式) {
    ...
}
if (判断表达式) {
    
} else if (判断表达式) {
    
} else {
    
}
switch (变量常量) {
    case ...: ....
}

最新的JDK可能因为加了其他关键字，可以结合switch做出其他花样，但基本用的也不多。所以还是和C一样。

3.5 循环语句

主要的循环有：

  • for
  • while
  • do-while

和C语言的一样，就是循环条件的结果一定要是boolean类型，而不是像C语言的非零即真。

例子：使用for循环完成$$1+2+3+…+100$$的结果。

public class Main {
  public static void main(String[] args) {
    int sum = 0;
    for (int i = 1; i <= 100; i++) {
      sum += i;
    }
    System.out.println("sum = " + sum);
  }
}

3.6 break和continue语句

基本用法和C一样。还有比较少用的用法，就是Java中保留了goto的用法，但是去掉了goto关键字，可以使用break和continue加标签的方式实现goto的效果，这里也就不演示了，因为一般也不用。

3.7 for语句与数组

JDK5对for的功能给予了扩充、增强，以便用户更好地遍历数组。其语法格式如下：

for (声明循环变量:数组的名字) {
  ...
}

其中，声明的循环变量的类型必须和数组的类型相同。这种形式的for语句类似自然语言中的for each语句，为了便于读者理解上述for语句，可以将这种形式的for语句翻译成”对于循环变量依次取数组中的每一个元素的值“。

例子：

public class Main {
  public static void main(String[] args) {
    int a[] = {3, 4, 2, 1};
    for (int i : a) {
      System.out.print(i + " ");
    }
    System.out.println();
  }
}

需要特别注意的是，”声明循环变量“必须是变量的声明，不可以使用已经声明过的变量。例如：

int a[] = {3, 4, 2, 1};
int i = 0;
for (i : a) {
  System.out.print(i + " ");
}

上述的代码是错误的。

3.8 小结

  1. Java提供了丰富的运算符，例如算术运算符、关系运算符、逻辑运算符、位运算符等
  2. Java语言中常用的控制语句和C语言中的很相似
  3. Java提供了遍历数组的循环语句
