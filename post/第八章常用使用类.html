8.1 String类

用于管理字符串的类。类似C++的string类

8.1.1 构造String对象

String常量

String常量也是对象，是用双引号括起来的字符序列。Java把用户程序中的String常量放入常量池。因为String常量是对象，所以也有自己的引用和实体。

注意：可以这样简单地理解常量池，常量池中的数据在程序运行期间再也不允许改变。

String对象

可以使用String类声明对象并创建对象。例如：

String s = new String("Hello");

对象变量s中存放着引用，表明自己的实体的位置，即new运算符首先分配内存空间并在内存空间中存放字符序列。

可以查看引用地址确认String类的实际存储位置：

int idetityHashCode(Object object);

String类常用的构造方法

// 1
char[] a = {'H', 'e', 'l', 'l', 'o'};
String s = new String(a);
// 相当于 String s = new String("Hello");

// 2
char[] b = {'W', 'o', 'r', 'l', 'd'};
String s2 = new String(b, 1, 3);  // orl

引用String常量

常量实际地址是不变的，当使用两个String对象相互赋值，其实是将常量池的数据的地址赋值。

不可变对象

所谓不可变对象是指对象的变量中存储的值不能再发生改变。String对象属于不可变变量，即String对象的实体无法再发生变化，其原因是String类是final类，String也没有给其对象提供修改实体的方法。

8.1.2 String对象的并置

就是字符串连接。

String hello = "Hello";
String world = "World";
String helloworld = hello + world; // helloworld获得的是一个新的地址，而不是由hello的地址上增加内容

注意：程序更关心两个String对象的实体，而不是二者的引用是否相同。

8.1.3 String的常用方法

  1. length 获取字符串长度。
String hello = "Hello你好";
int len = hello.length(); // 7
  1. equals 判断两个字符串
String tom = new String("天道酬勤");
String boy = new String("知心朋友");
String jerry = new String("天道酬勤");
boolean b = tom.equals(boy); // false
b = tom.equals(jerry);  // true

注意：tom==boy这种方式是比较这两个引用是否相等，而不是比较字符串是否相等。

  1. startsWith 判断字符串是否以什么开头
String hello = "Hello";
boolean b = hello.startsWith("Hel"); // true
b = hello.startsWith("he"); // false
  1. endsWith 判断字符串是否以什么结尾
String hello = "Hello";
boolean b = hello.endsWith("lo"); // true
b = hello.endsWith("ll"); // false
  1. compareTo 比较两个字符串
String s1 = "Hello";
String s2 = "World";
String s3 = "ABC";
String s4 = "Hello";
int i = s1.compareTo(s2); // 正数
i = s1.compareTo(s3); // 负数
i = s1.compareTo(s4); // 0
  1. contains 判断字符串是否包含某个字符串
String s1 = "Hello";
String s2 = "World";
String s3 = "ell";
boolean b = s1.contains(s2); // false
b = s1.contains(s3);  // true
  1. indexOf 从头开始找，获取字符在字符串的位置
  2. lastIndexOf 从尾开始找，获取字符在字符串的位置
  3. substring 获取字符串的子串，新建内存
String tom = "我喜欢音乐";
String str = tom.substring(2, 4); // 欢音
  1. trim 去掉字符串的前后空格，原字符串不变，返回新的String

8.2 正则表达式

8.2.1 正则表达式与元字符

正则表达式是一个String对象的字符序列，该字符序列中含有具有特殊意义的字符，这些特殊字符称为正则表达式中的元字符。

String对象调用public boolean matches(String regex)方法可以判断当前String对象的字符序列是否和参数regex指定的正则表达式匹配。

元字符意义
.代表任何字符
\..
\d0~9的数字
\D非数字字符
\s空格类等，如’\t’、’\n’等
\S非空格类
\w标识符字符（字母、数字、下划线）
\W非标识符字符
\p{Lower}小写字母
\p{Upper}大写字母
\p{ASCII}ASCII字符
\p{Alpha}字母（包括大写和小写）
\p{Digit}数字字符
\p{Alnum}字母或数字
\p{Punct}标点符号
\p{Graph}可视字符，包括\p{Alnum}、\p{Punct}
\p{Print}可打印字符
\p{Blank}空格或制表符
\p{Cntrl}控制字符 [\x00-\x1F\x7F]

在正则表达式中可以用方括号括起来若干个字符表示一个元字符，该元字符代表方括号中的任何一个字符。

  • [abc]：代表a、b、c中的任何一个
  • [^abc]：代表除了a、b、c以外的任何字符
  • [a-zA-Z]：代表英文字母中的任何一个
  • [a-d]：代表a~d中的任何一个

另外，方括号里允许嵌套方括号，可以进行并、交、差运算。

  • [a-d[m-p]]：代表ad或mp中的任何字符
  • [a-z&&[def]]：代表d、e或f中的任何一个
  • [a-f&&[^bc]]：代表a、d、e、f

注意由于.代表任何一个字符，所以在正则表达式中如果想要使用普通意义的.，必须使用[.]或\\.表示

例子：

String regex = "hello[1234]";

那么以下的字符串都可以成功匹配。

  • hello1
  • hello2
  • hello3
  • hello4

在正则表达式中可以使用限定修饰符。

带限定修饰符的模式意义
X?X出现0次或1次
X*X出现0次或多次
X+X出现1次或多次
X{n}X恰好出现n次
X{n, }X至少出现n次
X{n, m}X出现n次至m次
XYX的后缀是Y
X|YX或Y

例如：regex = "@\\w{4}"，那么”@abcd”、”@java”都可以匹配。

public class Main {
    public static void main(String[] args) {
        String regex = "@\\w{4}";
        String str = "@天道酬勤";
        boolean b = str.matches(regex);
        System.out.println(b);
        b = "@java".matches(regex);
        System.out.println(b);
    }
}
/*
false
true
*/

注意：有关正则表达式的细节可以查阅java.util.regex包中的Pattern类。

8.2.4 字符序列的分解

String类提供了一个实用的方法public String[] split(String regex)，String对象调用该方法时使用参数指定的正则表达式regex作为分隔标记分解出当前String对象的字符序列中的单词，并将分解出的单词存放在String数组中。例子：

public class Main {
    public static void main(String[] args) {
        String str = "1949年10月1日是中华人民共和国成立的日子";
        String regex = "\\D+";
        String[] digiWord = str.split(regex);
        for (var v : digiWord) {
            System.out.println(v);
        }
    }
}
/*
1949
10
1
*/

需要特别注意的是，split方法认为分隔标记的左右是单词，额外规则是如果左面是不含任何字符的字符序列（长度为0的字符序列，即””），这个字符序列仍然算是一个单词，但右边的单词必须是含有字符的字符序列。

例子：

public class Main {
    public static void main(String[] args) {
        String str = "公元1949年10月1日是中华人民共和国成立的日子";
        String regex = "\\D+";
        String[] digiWord = str.split(regex);
        for (var v : digiWord) {
            System.out.println(v);
        }
        System.out.println(digiWord.length);
    }
}
/*

1949
10
1
4
*/

8.3 StringTokenizer类

当分析一个String对象的字符序列并将字符序列分解成被独立使用的单词时，可以使用java.util包中的StringTokenizer类，该类有两个常用的构造方法。

  • StringTokenizer(String s)：为String对象s构造一个分析器。使用默认的分隔标记，即空格符、换行、回车符、制表符、进纸符做分隔标记。
  • StringTokenizer(String s, String delim)：为String对象s构造一个分析器。参数delim的字符序列中的字符的任意排列被作为分隔标记。

例子：

StringTokenizer fenxi = new StringTokenizer("you are welcome");
StringTokenizer fenxi = new StringTokenizer("you#*are*##welcome", "#*");
import java.util.StringTokenizer;

public class Main {
    public static void main(String[] args) {
        StringTokenizer fenxi = new StringTokenizer("you are welcome");
        while (fenxi.hasMoreTokens()) {
            System.out.println(fenxi.nextToken());
        }
        fenxi = new StringTokenizer("you#*are*##welcome", "#*");
        System.out.println("-------------------------------");
        while (fenxi.hasMoreTokens()) {
            System.out.println(fenxi.nextToken());
        }
        fenxi = new StringTokenizer("you are welcome(thank you), nice to meet you", "(),");
        System.out.println("-------------------------------");
        while (fenxi.hasMoreTokens()) {
            System.out.println(fenxi.nextToken());
        }
    }
}
/*
you
are
welcome
-------------------------------
you
are
welcome
-------------------------------
you are welcome
thank you
 nice to meet you
*/

一个分析器可以使用nextToken方法逐个获取分析器中的单词。分析器调用nextToken方法逐个获取分析器中的单词，每当nextToken返回一个单词，分析器就自动从分析器中删除该单词。类似队列。

例子：

import java.util.StringTokenizer;

class PriceToken {
    public double getPriceSum(String shoppingReceipt) {
        // 匹配非数字字符序列, 并用#代替, 然后再通过#拆分，这样就可以把数字提取出来
        StringTokenizer fenxi = new StringTokenizer(shoppingReceipt.replaceAll("[^0-9.]+", "#"), "#");
        double sum = 0;
        while (fenxi.hasMoreTokens()) {
            sum += Double.parseDouble(fenxi.nextToken());
        }
        return sum;
    }

    public int getGoodsAmount(String shoppintReceipt) {
        shoppintReceipt = shoppintReceipt.replaceAll("[^0-9.]+", "#");
        StringTokenizer fenxi = new StringTokenizer(shoppintReceipt, "#");
        return fenxi.countTokens();
    }

    public double getAverPrice(String shoppingReceipt) {
        return getPriceSum(shoppingReceipt) / getGoodsAmount(shoppingReceipt);
    }
}

public class Main {
    public static void main(String[] args) {
        String shoppingReceipt = "牛奶：8.5元，香蕉：3.6元，酱油：2.8元。";
        PriceToken lookPriceMess = new PriceToken();
        System.out.println(shoppingReceipt);
        double sum = lookPriceMess.getPriceSum(shoppingReceipt);
        System.out.printf("购物总价格为：%-7.2f", sum);
        int amount = lookPriceMess.getGoodsAmount(shoppingReceipt);
        double aver = lookPriceMess.getAverPrice(shoppingReceipt);
        System.out.printf("\n商品数目为：%d, 平均价格为：%-7.2f", amount, aver);
    }
}
/*
牛奶：8.5元，香蕉：3.6元，酱油：2.8元。
购物总价格为：14.90  
商品数目为：3, 平均价格为：4.97 
*/

上面的代码很长，但其实内容很简单，就是把非数字字符和非小数点的用一个临时符号代替（上面用的#），然后再通过#拆分，最后就剩下了数字，然后按正常程序打印。

8.2 Scanner类

本节学习怎样使用Scanner类的对象从字符序列中解析出程序需要的数据。

  1. Scanner对象
String Musice = "I love this art";

为了解析出Musice的字符序列中的单词，可以如下构造一个Scanner对象：

Scanner scanner = new Scanner(Musice);

Scanner对象调用方法：

useDelimiter(正则表达式); // 使用分隔符

将正则表达式作为分隔标记。如果不指定分隔标记，那么Scanner对象默认用空白字符作为分隔标记来解析String对象的字符序列中的单词。

  • Scanner对象调用next方法依次返回被解析的单词，如果最后一个单词已被next方法返回，Scanner对象调用hasNext将返回false，否则返回true。
  • 对于被解析的字符序列中的数字型单词，Scanner对象可以使用nextInt、nextDouble等方法。
  • 如果单词不是数字型单词，Scanner对象调用nextInt或nextDouble方法将发生InputMismatchException异常，在处理异常时可以调用next方法返回非数字化单词。

例子：

import java.util.InputMismatchException;
import java.util.Scanner;

class GetPrice {
    public static double givePriceSum(String cost) {
        Scanner scanner = new Scanner(cost);
        scanner.useDelimiter("[^0-9.]+");
        double sum = 0;
        int count = 0;
        while (scanner.hasNext()) {
            try {
                sum += scanner.nextDouble();
            } catch (InputMismatchException e) {
                System.out.printf("发生了%d次异常\n", ++count);
                String t = scanner.next();
            }
        }
        return sum;
    }
}

public class Main {
    public static void main(String[] args) {
        String cost = "市话76.8元，长途167.38元，短信12.68元";
        double priceSum = GetPrice.givePriceSum(cost);
        System.out.printf("%s\n总价：%.2f\n", cost, priceSum);
        cost = "牛奶8.5元，香蕉3.6元，酱油2.8元";
        priceSum = GetPrice.givePriceSum(cost);
        System.out.printf("%s\n总价：%.2f\n", cost, priceSum);
    }
}
/*
市话76.8元，长途167.38元，短信12.68元
总价：256.86
牛奶8.5元，香蕉3.6元，酱油2.8元
总价：14.90
*/
  1. StringTokenizer和Scanner的区别

StringToKenizer类和Scanner类都可用于分解字符序列中的单词，但二者在思想上有所不同。

类名说明
StringTokenizer把分解出的全部单词都放到StringTokenizer对象实体中，因此StringTokenizer对象能以较快速度获得单词（空间换时间）
Scanner不把单词存放到Sanner对象的实体中，而是仅存放怎样获取单词的分隔标记，因此Scanner对象获得单词的速度相对较慢，但Scanner对象节省内存空间（时间换空间）

8.5 Pattern类与Matcher类

模式匹配就是检索和指定模式匹配的字符序列。Java提供了专门用来进行模式匹配的Pattern类和Matcher类，这些类在java.util.regex包中。

使用Pattern类和Matcher类的步骤。

  1. 数据源
String str = "hello,good morning,this is a good idea";
  1. 建立Pattern类的对象

假设str中从哪个位置开始至那个位置结束曾出现了字符序列good，那么首先将用于检索数据的正则表达式封装在一个Pattern对象中，需要按下列方式使用正则表达式regex做参数得到一个称为模式的Pattern类的实例。

Pattern pattern = Pattern.compile(regex);
  1. 得到Matcher对象

得到用于检索数据源的Matcher类的实例。

Matcher matcher = pattern.matcher(str);
  1. 检索数据

匹配对象matcher就可以调用各种方法检索数据源，即str中的数据，例如，matcher依次调用boolean find()方法可以检索str的字符序列中和regex匹配的子字符序列。

  1. Matcher类提供的常用方法

Matcher类的对象matcher可以使用下列方法寻找String对象str和regex匹配的子序列。

方法名说明
find寻找str和regex匹配的下一子序列，如果匹配成功则返回true，否则返回false。
matches判断str是否完全和regex匹配
lookingAt判断从str的开始位置是否和regex匹配的子序列。
replaceAll可以返回一个String对象，该String对象的字符序列是通过把str的字符序列中与模式regex匹配的子序列全部替换为参数replacement指定的字符序列得到的
replaceFist可以返回一个String对象，该String对象的字符序列是通过把str的字符序列中第一个与模式regex匹配的子序列替换为参数replacement指定的字符系列得到的

例子：

import java.util.regex.Matcher;
import java.util.regex.Pattern;

public class Main {
    public static void main(String[] args) {
        String str = "市话76.8元，长途167.38元，短信12.68元，其他20元";
        String regex = "-?[1-9][0-9]*[.]?[0-9]*";   // 匹配整数或浮点数
        Pattern pattern = Pattern.compile(regex);   // 模式对象
        Matcher matcher = pattern.matcher(str);     // 匹配对象
        double sum = 0;
        while (matcher.find()) {
            String item = matcher.group(); // 返回与前一个匹配项匹配的输入子序列。
            System.out.println(item);
            sum += Double.parseDouble(item);
        }
        System.out.println("账单总价格：" + sum);
    }
}

167.38
12.68
20
账单总价格：276.86
*/

8.6 StringBuffer类

String对象的字符序列是不可修改的，也就是说，String对象的字符序列的字符不能被修改、删除，即String对象的实体是不可以再发生变化。例如，对于

String s = new String("我喜欢散步");

引用s可以改变指向，但实体String(“我喜欢散步”)是不可以再发生变化。

与String类不同，StringBuffer类的对象的实体的内存空间可以自动改变大小，便于存放一个可变的字符序列。尽管StringBuffer类是java.lang包中的final类，但该类提供了修改字符序列的方法。

StringBuffer strBuffer = new StringBuffer("我喜欢");

对象stringBuffer可调用append()方法追加一个字符序列，

strBuffer.append("玩篮球"); // 在原来的基础上增加了"玩篮球",地址不变

StringBuffer类有3个构造方法，即

  • StringBuffer() 创建一个StringBuffer对象，初始容量可以容纳16个字符，当该对象的实体存放的字符序列的长度大于16时，实体的容量自动增长，以便存放所增加的字符
  • StringBuffer(int size) 可以指定分配给该对象的实体的初始容量
  • StringBuffer(String s) 可以指定分配该对象的实体的初始容量为参数s的字符序列的长度再加16

StringBuffer对象可以通过调用length()方法获取实体中存放的字符序列的长度，通过capacity()方法获取当前实体的实际容量。

8.7.2 日期、时间差和日历类

在许多应用中可能经常需要计算两个日期或时间的差。知道有这么一回事就可以。

long until(Temporal endExclusive, TemporaUnit unit);

例如：

dataStart的日期是2021-2-4，dataEnd的日期是2022-7-9，那么

long n;
n = dataStart.until(dataEnd, ChroundUnit.DAYS);   // n == 520
n = dataStart.until(dataEnd, ChroundUnit.MONTHS); // n == 17
n = dataStart.until(dataEnd, ChroundUnit.YEARS);  // n == 1

LocalDate、LocalDateTime、LocalTime类都是实现了Temporal接口的类。枚举类型ChronoUnit实现了TemporalUnit接口，提供了许多枚举常量。

注意：如果dateEnd小于startDate，until方法返回的是负数。在计算日期差时不足一个单位的零头按0计算。

例子：

import java.time.LocalDateTime;
import java.time.temporal.ChronoUnit;

public class Main {
    public static void main(String[] args) {
        LocalDateTime dateStart = LocalDateTime.of(1931, 9, 18, 0, 0, 0);
        LocalDateTime dateEnd = LocalDateTime.of(1945, 8, 15, 0, 0, 0);
        long years = dateStart.until(dateEnd, ChronoUnit.YEARS);
        long months = dateStart.until(dateEnd, ChronoUnit.MONTHS);
        long days = dateStart.until(dateEnd, ChronoUnit.DAYS);
        long hours = dateStart.until(dateEnd, ChronoUnit.HOURS);
        long weeks = dateStart.until(dateEnd, ChronoUnit.WEEKS);
        System.out.println(dateStart + "和" + dateEnd + "相差：");
        System.out.println(years + "年");
        System.out.println(months + "个月");
        System.out.println(days + "日");
        System.out.println(hours + "个小时");
        System.out.println(weeks + "个星期");
        LocalDateTime nextDateStart = dateStart.plusYears(years);
        months = nextDateStart.until(dateEnd, ChronoUnit.MONTHS);
        nextDateStart = nextDateStart.plusMonths(months);
        days = nextDateStart.until(dateEnd, ChronoUnit.DAYS);
        System.out.println(dateStart + "和" + dateEnd + "相差：");
        System.out.println(years + "年" + months + "个月" + days + "天");

    }
}

8.7.3 日期格式化

有时候可能希望按照某种习惯来输出时间。

年 月 星期 日
------------
年 月 星期 日 小时 分 秒
  1. format方法
String format(格式化模式, 日期列表)

返回一个String对象，该对象的字符序列是把“格式化模式”的格式符号替换成“日期列表”中对应数据（年、月、日、小时等数据）后的字符序列。

format方法中的“格式化模式”是一个用双引号括起来的字符序列，该字符序列中的字符由时间格式符和普通字符所构成。

"日期：%ty-%tm-%td"

其中的%ty、%tm、%td等都是时间格式符，等于的都是普通字符。

format方法返回的String对象的字符序列就是把“格式化模式”中的时间格式符替换为相应的时间后的字符序列。

LocalDate date = LocalDate.now();
String s = String.format("%ty年%tm月%td日", date, date, date);

那么s就是被格式化成xx年xx月xx日.

以下是常用的日期格式符表：

格式符作用
%tY将日期中的“年”格式化为4位形式
%ty将日期中的“年”格式化为2位形式
%tm将日期中的“月”格式化为2位形式(带前缀0)
%td将日期中的“日”格式化为当前月中的天（带前缀0）
%tB将日期中的“月”格式化为当前环境下的月份全称
%tb将日期中的“月”格式化为当前环境下的月份简称
%tA将日期中的“星期”格式化为当前环境下的星期全称
%ta将日期中的“星期”格式化为当前环境下的星期简称
%tH将日期中的“时”格式化为2位形式(带前缀0，24小时制)
%tI将日期中的“时”格式化为2位形式(带前缀0，12小时制)
%tM将日期中的“分”格式化为2位形式(带前缀0)
%tS将日期中的“秒”格式化为2位形式(带前缀0)
%tN将日期中的“纳秒”格式化为2位形式(带前缀0)
%tR等价%tH:%tM
%tT等价%tH:%tM:%tS
%tr等价%tI:%tM:%tS%tp（上午或下午）
%tD等价于%tm/%td/%ty
%tF等价于%tY-%tm-%td
%tc等价于%ta %tb %td %tT %tZ %tY
  1. 不同区域的星期格式

不同国家的星期的简称或全称有很大的不同。如果想用特定地区的星期格式来表示日期中的星期，可以用format的重载方法。

format(Local local, 格式化模式, 日期列表)

其中参数local是一个Local类的实例，用于表示地域。

例子：

import java.time.LocalDateTime;
import java.util.Locale;

public class Main {
    public static void main(String[] args) {

        String pattern = "%tY-%<tm-%<td, %<tA, %<tT";
        String s = String.format(pattern, nowTime);
        System.out.println(s);
        s = String.format(Locale.US, pattern, nowTime);
        System.out.println(s);
        s = String.format(Locale.JAPAN, pattern, nowTime);
        System.out.println(s);
    }
}

8.8.3 Random类

尽管可以调用Math类中的static方法random返回一个0~1的随机数（包括0.0但不包括1.0），即随机数的取值范围是[0.0, 1.0).

(int)(Math.random()*100)+1; // [1, 100]

因为random不会随机到1.0，所以上述的(Math.random()*100)不可能得到100.0，最多只能达到99.9999…，然后通过强制类型转换为整型就会变成99，然后再加后面的1就可以达到100，所以上面的代码的结果就是[1, 100]之间的整数。

但是，Java提供了更为灵活的用于获取随机数的Random类（该类在java.util包中）。

使用Random类的如下构造方法：

public Random();    // 使用当前机器时间作为种子创建一个Random对象
public Random(long seed); // 使用参数seed指定的种子创建一个Random对象

人们习惯将Random对象称为随机数生成器。

例子：

Random random = new Random();
random.nextInt(); // 返回一个随机整数

如果想让随机数生成器random返回一个0~n（包括0不包括n）的随机数，可以让random调用参数的nextInt(int m)方法（参数m必须取正整数）。

例子：

random.nextInt(100); // 返回[0, 99]的某个整数

random调用public double nextDouble()返回一个0.0~1.0（包括0.0不包括1.0）的随机数。

如果程序需要随机得到true和false两个表示真和假的boolean值，可以用random调用nextBoolean方法。

注意：对于具有相同种子的两个Random对象，二者依次调用nextInt()方法获取的随机数序列是相同的。

例子：

随机获得6个1~100的不同的整数。

import java.util.Random;

class GetRandomNumber {
    public static int[] getRandomNumber(int max, int amount) {
        // 1~max随机整数
        int[] randNumber = new int[amount];
        int index = 0;
        randNumber[0] = -1;
        Random random = new Random();
        while (index < amount) {
            int number = random.nextInt(max) + 1;   // [1, max]
            boolean isInArrays = false;
            for (int m : randNumber) {
                if (m == number) {
                    isInArrays = true;
                    break;
                }
            }
            if (!isInArrays) {
                randNumber[index++] = number;
            }
        }
        return randNumber;
    }
}

public class Main {
    public static void main(String[] args) {
        int[] a = GetRandomNumber.getRandomNumber(100, 6);
        System.out.println(java.util.Arrays.toString(a));
    }
}
/*
[79, 96, 92, 15, 73, 81]
*/

            saveNumber[i] = i + 1;
        }
        int[] frequency = new int[number];
        Random random = new Random();
        int counts = 10000;
        int i = 1;
        while (i < counts) {
            int m = random.nextInt(number) + 1;
            int index = Arrays.binarySearch(saveNumber, m);
            if (index >= 0) {
                frequency[index]++;
            }
            i++;
        }
        System.out.println("循环" + counts + "次");
        System.out.println(Arrays.toString(saveNumber));
        System.out.println("各个数字出现的次数：");
        System.out.println(Arrays.toString(frequency));
        int sum = 0;
        for (int item : frequency) {
            sum += item;
        }
        System.out.println("次数之和sum = " + sum);
    }
}
/*
循环10000次
[1, 2, 3, 4, 5, 6, 7]
各个数字出现的次数：
[1419, 1395, 1486, 1476, 1367, 1395, 1461]
次数之和sum = 9999
*/

8.8.4 数字格式化

程序有时候需要对数字进行格式化。所谓数字格式化，就是按照指定格式得到一个字符序列。

  1. 格式化模式

format(格式化模式, 值列表)方法中的”格式化模式“是一个用双引号括起来的字符序列，该字符序列中的字符由格式符合普通字符构成。

  • %d：将值格式化为十进制整数
  • %o：将值格式化为八进制整数
  • %x：将值格式化为小写的十六进制整数
  • %X：将值格式化为大写的十六进制整数
  • %f：将值格式化为十进制浮点数，小数保留6位
  • %e（%E）：将值格式化为科学记数法的十进制的浮点数

规定宽度的一般格式为：%mf或%md（在数字的前面增加空格）

public class Main {
    public static void main(String[] args) {
        String s = String.format("%11f", 59.88);
        System.out.println("输出一句话");
        System.out.println(s);
    }
}
/*
输出一句话
  59.880000
*/
// 11 = 2 + 2 + 1 + 6
// m  = (空格) + (整数部分) + (小数点) + (小数部分)

如果m是负数则，从右边填充空格。

在指定宽度的同时也可以限制小数位数（%m.nf），表示这个数字总共占m位，小数部分占n位。

  1. 值列表

format方法的”值列表“是用都好分隔的变量、常量或表达式，要保证format方法的”格式化模式“中的格式符的个数与”值列表“中的值的个数相同。

String s = String.format("%d元%0.3f公斤%d台", 888, 999.777666, 123);
// s == "888元999.778公斤123台"

注意：如果准备在”格式化模式“中包含普通的%，在编写代码时需要连续输入两个%.

8.9 Class类与反射

8.9.1 Java反射

Class是java.lang包中的类，Class的实例封装和类有关的信息。任何类都默认有一个public的静态的Class对象，该对象的名字是class（用关键字做了名字，属于Java系统特权），该对象封装当前类的有关信息。用户可以通过类的对象调用getClass()方法（从java.lang.Object类继承的方法）返回这个Class对象——class。

Class对象调用如下方法可以获取当前类的有关信息，例如类的名字、类中的方法名称、成员变量的名称等，这一机制也称为Java反射。

  • String getName()：返回类的名字
  • Constructor[] getDeclaredConstructors()：返回类的全部构造方法
  • Field[] getDeclaredFields()：返回类的全部成员变量
  • Method[] getDeclaredMethods()：返回类的全部方法。

例子：

import java.lang.reflect.Constructor;
import java.lang.reflect.Field;
import java.lang.reflect.Method;

class Rect {
    double width;
    double height;
    double area;

    public Rect() {
    }

    public Rect(double width, double height) {
        this.width = width;
        this.height = height;
    }

    public double getArea() {
        area = height * width;
        return area;
    }
}

public class Main {
    public static void main(String[] args) {
        Rect rect = new Rect();
        Class cs = rect.getClass();
        System.out.println(cs == Rect.class);
        String className = cs.getName();
        Constructor[] constructors = cs.getDeclaredConstructors();   // 返回类中的构造方法
        Field[] fields = cs.getDeclaredFields();                     // 返回类中的成员变量
        Method[] methods = cs.getDeclaredMethods();                  // 返回类的方法
        System.out.println("类的名字: "+className);
        System.out.println("类中有如下的构造方法: ");
        for (Constructor constructor : constructors) {
            System.out.println(constructor.toString());
        }
        System.out.println("类中有如下的成员变量:");
        for (Field field : fields) {
            System.out.println(field.toString());
        }
        System.out.println("类中有如下方法");
        for (Method method : methods) {
            System.out.println(method.toString());
        }
    }
}
/*
true
类的名字: Rect
类中有如下的构造方法: 
public Rect()
public Rect(double,double)
类中有如下的成员变量:
double Rect.width
double Rect.height
double Rect.area
类中有如下方法
public double Rect.getArea()
*/

8.9.2 使用Class实例化一个对象

得到一个类的实例的常用方式就是使用new，在Java中也可以使用和类相关的Class对象(class)得到一个类的实例。

为了使用Class得到一个类的实例，可以先得到一个和该类相关的Class对象，做到这一点并不困难。但下面的方式更加灵活。

public static Class<?> forName(String className) throws ClassNotFoundException

就可以返回一个和参数className指定的类相关的Class对象。再让这个Class对象调用方法得到className的无参构造方法（因此className类必须保证有无参构造方法）。

然后，Constructor<?>对象调用newInstance()返回一个className类的对象。

public Constructor<?> getDeclaredConstructor() throws SecurityExcepthon

注意：JDK9版本开始，Class类的newInstance()方法被宣布过时。

例子：

编译器只检查Circle是否是一个String类型的数据，并不检查是否是Circle类import java.lang.reflect.Constructor;

class Circle {
    public double radius;

    public void setRadius(double radius) {
        this.radius = radius;
    }

    public double getArea() {
        return Math.PI * radius * radius;
    }
}

public class Main {
    public static void main(String[] args) {
        try {
            Class<?> cs = Class.forName("Circle");  // 
            Constructor<?> constructor = cs.getDeclaredConstructor();   // 获得构造器（构造方法）
            Circle circle = (Circle) constructor.newInstance();         // 通过刚刚的构造器new一个实例
            circle.setRadius(100);
            double area = circle.getArea();
            String formatStr = String.format("%10.2f", area);
            System.out.println("circle的半径: " + circle.radius);
            System.out.println("circle的面积: " + area);
        } catch (Exception e) {
            System.out.println(e.getMessage());
        }
    }
}
/*
circle的半径: 100.0
circle的面积: 31415.926535897932
*/

在forName那行，编译器只检查Circle是否是一个String类型的数据，并不检查是否是Circle类。在运行阶段，如果发现没有Circle类，将触发ClassNotFoundException.而对于”Circle.class”在编译阶段就要检查是否有Circle类（缺乏灵活性）。

注意：在后面学习数据库时，经常需要使用Class类加载和数据库驱动相关的类，纯Java数据库驱动都是一个Java类。

8.10 Arrays类、System类和Console类

8.10.1 Arrays类

Arrays在java.util包中，提供了操作数组的各种static方法。

方法作用
public staic void sort(double[] a)对参数指定的double型数组a按数字升序进行排序
public static int binarySearch(double[] a, double key)使用二分查找判断key给出的值是否在参数a指定的数组中，在则返回其下标，否则返回-1
public static boolean equals(double[] a, double[] b)判断两个数组是否相等，是则返回true，否则返回false
public static String toString(double[] a)将数组a的数据转为字符串用逗号分隔，再用一个[]包裹
public static double[] copyOf(double[] original, int newLength)复制数组原始，新数组的长度为max(newLength, original.length)，默认用0.0填充

上面的方法是Arrays类的重载方法，当然还有其它类型的版本，这里不一一列出。

8.10.2 System类

System类在java.lang包中，下列static方法属于常用方法。

方法作用
public static long nanoTime()以纳秒为单位返回最准确的可用系统计时器的当前值。此方法只能用于测量已过的时间，与系统或钟表时间的其他任何时间概念无关
public static void exit(int status)终止当前正在运行的Java虚拟机，即退出当前Java虚拟机正在运行的全部程序
public static void gc()运行垃圾回收器

例子：

import java.util.Arrays;
import java.util.Random;

public class Main {
    public static void main(String[] args) {
        Random randomOne = new Random(2023);
        Random randomTwo = new Random(2023);
        int amount = 10000;
        int[] one = new int[amount];
        int[] two = new int[amount];
        for (int i = 0; i < amount; i++) {
            one[i] = randomOne.nextInt(amount);
            two[i] = randomOne.nextInt(amount);
        }
        if (Arrays.equals(one, two)) {
            System.out.println("两个数组相等，前8个元素分别是：");
            int[] a = Arrays.copyOf(one, 8);
            int[] b = Arrays.copyOf(two, 8);
            System.out.println(Arrays.toString(a));
            System.out.println(Arrays.toString(b));
        } else {
            System.out.println("两个数组不相等");
        }
        long startTime = System.nanoTime();
        Arrays.sort(one);
        Arrays.sort(two);
        long elapsedNanos = System.nanoTime() - startTime;
        System.out.println("排序两个数组的用时(纳秒)" + elapsedNanos);
    }
}
/*
两个数组不相等
排序两个数组的用时(纳秒)6730500
*/

8.10.3 Console类

如果希望在键盘上输入一行文本，但不想让该文本回显，即不在命令行中显示，那么需要使用java.io包中的Console类的对象来完成。首先使用System类调用console()方法返回Console类的一个对象。

Console console = System.console();

然后console调用readPassword()方法读取用户在键盘输入的一行文本，并将文本以一个char型数组返回。

char[] passwd = console.readPassword();

例子：

import java.io.Console;

public class Main {
    public static void main(String[] args) {
        boolean success = false;
        int count = 0;
        Console cons;
        char[] passwd;
        cons = System.console();
        while (true) {
            System.out.print("输入密码：");
            passwd = cons.readPassword();
            count++;
            String password = new String(passwd);
            if (password.equals("I Love You")) {
                success = true;
                System.out.println("第" + count + "次密码正确!");
                break;
            } else {
                System.out.println("第" + count + "次密码" + password + "不正确");
            }
            if (count == 3) {
                System.out.println("您" + count + "次输入的密码都不正确");
                System.exit(0);
            }
        }
        System.out.println("您好，登入成功");
    }
}

程序出现异常。。。
*/

8.11 小结

  1. 熟练掌握String类的常用方法，这些方法对于有效处理字符序列信息是非常重要的

  3. 使用StringTokenizer、Scanner、Pattern、Matcher类分析字符串，获取字符串中的单词
  4. 熟悉Class类的机制和用法
  5. 当程序需要处理时间时，使用java.time包中的LocalDate、LocalDateTime和LocalTime类
  6. 如果需要处理特别大的整数，使用BigInteger类
  7. 当需要格式化日期和数字时，使用String类的static方法format
