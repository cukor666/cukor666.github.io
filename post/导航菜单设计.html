导航菜单设计

导航菜单是网站重要的组成部分。导航菜单的设计关系着网站的可用性和用户体验，有吸引力的导航能够吸引用户去浏览更多的网站内容。设计一个优秀的页面导航菜单会给网站增色不少。作为一名Web前端开发工程师必须掌握传统的网站导航菜单设计技巧，同时也需要学习响应式导航菜单的设计方法。

网站菜单表现形式丰富多彩。从菜单层次上看，可以分为一级、二级和多级菜单。从排列方式上看，可以分水平导航、垂直导航菜单。从技术实现角度上看，导航菜单通常采用无序列表、表格、超链接和样式表相结合的方法来实现，也可以使用如CSS3 Menu、jQuery等第三方插件技术来实现。

一级水平导航菜单

采用“表格+超链接”来设计

使用表格布局设计一级导航菜单非常容易而且布局均匀，根据导航栏目数量确定表格的列数。采用1行10列表格，第1、10单元格插入空格，留出左右边空白，其余单元格插入超链接即可实现，代码如下：

<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        table {
            width: 680px;
            height: 40px;
            text-align: center;
            background: #ff6666;
        }

        a:link,
        a:visited,
        a:hover,
        a:active {
            text-decoration: none;
            color: white;
        }
        a:hover{
            color: yellow;
            border-bottom: 5px solid #ff0000;
        }
    </style>
</head>

<body>
    <table>
        <tr>
            <td>&nbsp;</td>
            <td><a href="">首页</a></td>
            <td><a href="">期刊介绍</a></td>
            <td><a href="">编委会</a></td>
            <td><a href="">常见问题及解答</a></td>
            <td><a href="">常用文档下载</a></td>
            <td><a href="">订阅</a></td>
            <td><a href="">过刊浏览</a></td>
            <td><a href="">优先出版</a></td>
            <td>&nbsp;</td>
        </tr>
    </table>
</body>

</html>

采用“无序列表+超链接”来设计

采用无序列表设计“一级水平导航菜单”需要做两件事：

  • 要去掉列表项前的符号
  • 将垂直显示的列表项转换为水平显示

二级水平导航菜单

商业网站上导航菜单一般有多种表现形式，分别是一级导航菜单、二级导航菜单、多种形式并存的导航菜单。

下拉导航菜单

借助于JavaScript设计网站下拉菜单的案例比较多见，采用纯CSS设计网站下拉菜单需要对样式进行详细定义才能实现。不过要考虑到不同浏览器之间的兼容性。下面列举一个仅仅采用<ul></ul>、<li></li>、<a></a>等标记和CSS样式定义来实现简单的二级下拉菜单的设计过程。

  1. 逐步设置样式

    （1）定义ul的样式，设置边距和填充均为0px.

    ul {
  margin: 0px;
  padding: 0px;
}

    （2）定义列表项样式，有垂直排列改为水平排列

    ul li {
  height: 30px;
  width: 115px;
  list-style: none;
  float: left;
  display: inline;
  font: 0.9em Arial;
}

    （3）定义超链接样式

    ul li a{
  color: white;
  width: 113px;
  margin: 0px;
  padding: 0px 0px 0px 8px;
  text-decoration: none;
  display: block;
  background: #808080;
  line-height: 29px;
  border-right: 1px solid #ccc;
  border-bottom: 1px solid #ccc;
}

    （4）定义嵌套列表项和子菜单超链接的规则

    ul li ul li {
  height: 25px;
}
ul li ul li a{
  background: #666;
  line-height: 24px;
}

    （5）定义鼠标滑过某个菜单时的样式

    ul li a:hover{
  background: #666;
  border-bottom: 1px dashed #ff0000;
}

    （6）定义子菜单项初始状态为隐藏

    ul li ul{
  display: none;
}

    （7）定义鼠标滑过时下拉子菜单显示样式

    ul li:hover ul {
  display: block;
}

ul li ul li a:hover {
  background: #333;
}

横向二级导航菜单

所谓横向二级导航菜单，就是一层主菜单是水平排列、二层子菜单也是水平排列各占一行，其中二层子菜单可能会占多行，取决于子菜单的数量。

采用CSS打造横线二级导航菜单，需要对HTML中的div、ul、li、a等标记进行样式定义，并应用样式。再设计下拉菜单的基础上，很容易实现横向二级导航菜单。

具体步骤如下：

  1. 设计HTML代码
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <link rel="stylesheet" href="level12_menu.css">
</head>

<body>
    <div id="menu">
        <ul>
            <li><a href="#">首页</a></li>
            <li><a href="#">jQuery特效</a>
                <div id="" class="submenu">
                    <ul>
                        <li><a href="#">jQuery图片特效</a></li>
                        <li><a href="#">jQuery导航特效</a></li>
                        <li><a href="#">jQuery选项卡特效</a></li>
                        <li><a href="#">jQuery文字特效</a></li>
                    </ul>
                </div>
            </li>
            <li><a href="#">Javascript特效</a></li>
            <li><a href="#">Flash特效</a>
                <div id="" class="submenu">
                    <ul>
                        <li><a href="#">Flash图片特效</a></li>
                        <li><a href="#">Flash导航特效</a></li>
                        <li><a href="#">Flash选项卡特效</a></li>
                        <li><a href="#">Flash文字特效</a></li>
                    </ul>
                </div>
            </li>
            </li>
            <li><a href="#">div+css教程</a></li>
            <li><a href="#">HTML5教程</a></li>
        </ul>
    </div>
</body>

</html>
  1. 定义HTML中相关标记的样式
#menu {
    padding-left: 100px;
    margin: 0 auto;
    text-align: center;
    width: 100%;
    height: 60px;
    background: #55aaee;
    border: 1px solid #333333;
}

#menu ul {
    margin: 0px;
    padding: 0px;
}

.submenu {
    width: 900px;
    height: 28px;
    text-align: center;
}

#menu ul li{
    height: 30px;
    width: 115px;
    list-style: none;
    float: left;
    display: inline;
    font: 0.9em Arial;
    text-align: center;
}
ul li a {
    color: #fff;
    width: 114px;
    margin: 0px;
    padding-left: 8px;
    text-decoration: none;
    display: block;
    background: #55a0ff;
    line-height: 29px;
    border-bottom: 1px solid #ccc;
}

ul li .submenu ul li {
    height: 25px;
}

ul li .submenu ul li a {
    background: #55aaee;
    line-height: 24px;
}

ul li a:hover {
    background: #ee6655;
    border-bottom: 1px dashed greenyellow;
}

ul li .submenu{
    display: none;
}

.submenu ul li {
    height: 24px;
    width: 113px;
    list-style: none;
    float: left;
    display: inline;
    font: 0.8em Arial;
    text-align: center;
}
ul li:hover .submenu {
    display: block;
}

ul li .submenu ul li a:hover {
    background: #333;
}
