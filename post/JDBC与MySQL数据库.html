JDBC与MySQL数据库

JDBC基本概念

JDBC是Java数据库连接（Java Database Connectivity）的缩写，它是一套用于执行SQL语句的Java API。应用程序可以通过这套API连接到关系数据库，并使用SQL语句来完成对数据库中数据的查询、更新和删除等操作。

JDBC提供了一种统一的、基于Java语言的关系数据库编程接口规范，该规范允许你把SQL语句作为参数通过JDBC接口发送给远端数据库，远端数据库接受到你的SQL后进行语法分析、验证，然后执行、响应。

连接MySQL数据库的准备

要使用Java连接MySQL数据库，你需要准备MySQL的驱动包。最新版本的驱动包可以从MySQL官网下载，下载地址为：http://dev.mysql.com/downloads/connector/j/。如下图所示（参考）。

下载并解压后，你会得到一个jar库文件，然后在对应的项目中导入该库文件即可。

使用MySQL驱动包

如果你已经下载好了MySQL的驱动包，那么你需要在对应的Java项目中导入该库文件。具体操作方法取决于你使用的开发工具。

Eclipse

例如，如果你使用的是Eclipse，那么你可以在项目中右键单击，选择 Build Path -> Configure Build Path -> Libraries -> Add External JARs，然后选择你下载的MySQL驱动包所在的位置，最后点击 Apply and Close 即可。

IntelliJ IDEA

如果你使用的是IntelliJ IDEA，那么你可以按照以下步骤来导入MySQL驱动包：

  1. 打开IDEA，选择 File -> Project Structure -> Modules。
  2. 选中右侧的 + 号，点击 JARs or directories。
  3. 选择你电脑中的MySQL驱动包所在位置。
  4. 导入这个jar包后，点击 Apply，再点击 OK 即可。

这样你就成功地在IntelliJ IDEA中导入了MySQL驱动包。

示例

如果你已经成功地将MySQL驱动包导入到你的项目中，那么你可以使用Java的JDBC API来连接MySQL数据库并执行SQL语句。

下面是一个简单的示例，演示如何使用Java连接MySQL数据库并执行查询操作：

package cn.cukor.jdbctest;

import java.sql.*;

public class Main {
    public static void main(String[] args) {
        // 数据库连接信息，根据自己的实际情况设置
        String url = "jdbc:mysql://localhost:3306/students";    // 实际数据库的地址
        String username = "root";        // 用户名
        String password = "123456";      // 密码

        // 加载驱动（以MySQL为例）
        try {
            Class.forName("com.mysql.cj.jdbc.Driver");
        } catch (ClassNotFoundException e) {
            e.printStackTrace();
        }

        // 连接数据库
        try (Connection conn = DriverManager.getConnection(url, username, password)) {
            // 创建Statement对象
            Statement stmt = conn.createStatement();

            // 执行查询操作
            ResultSet rs = stmt.executeQuery("select * from stu");

            // 处理查询结果
            System.out.println("sno\tname\tgander\tbirthday");
            while (rs.next()) {
                int sno = rs.getInt("sno");
                String name = rs.getString("name");
                String gander = rs.getString("gander");
                Date birthday = rs.getDate("birthday"); // java.sql中的Date
                System.out.printf("%d\t%s\t%s\t%s\n", sno, name, gander, birthday);
            }
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}

在上面的代码中，我们首先加载了MySQL的驱动，然后使用 DriverManager.getConnection() 方法来连接MySQL数据库。接着，我们创建了一个 Statement 对象，并使用它来执行SQL查询操作。最后，我们处理查询结果并输出结果。

你需要根据你自己的实际情况来修改上面代码中的数据库连接信息，包括数据库URL、用户名和密码。

这段代码使用了Java 7中引入的 try-with-resources 语句。这种语句可以确保在程序执行完毕后，自动关闭在括号中声明的资源。

在上面的示例代码中，我们在括号中声明了一个 Connection 对象。这意味着当程序执行完毕后，这个 Connection 对象会被自动关闭，无需我们手动调用 close() 方法来关闭它。

try-with-resources 语句可以让我们更方便地管理资源，避免因为忘记关闭资源而导致的问题。你可以在Java官方文档中了解更多关于 try-with-resources 语句的信息。

Connection类

Connection 类是Java的JDBC API中的一个接口，它表示一个到数据库的连接。你可以使用 Connection 对象来创建 Statement、PreparedStatement 和 CallableStatement 对象，用于执行SQL语句。

此外，Connection 类还提供了一些方法来管理事务。例如，你可以使用 setAutoCommit() 方法来设置是否自动提交事务，使用 commit() 方法来提交事务，使用 rollback() 方法来回滚事务。

总之，Connection 类是Java程序与数据库之间交互的核心接口之一。你可以使用它来执行SQL语句、管理事务等操作。

Statement类

Statement 类是Java的JDBC API中的一个接口，它表示一个用于执行SQL语句的对象。你可以使用 Statement 对象的 executeQuery() 方法来执行查询操作，使用 executeUpdate() 方法来执行插入、更新和删除操作。

Statement 对象通常用于执行静态SQL语句，即不带参数的SQL语句。如果你需要执行带参数的SQL语句，那么可以使用 PreparedStatement 类。

需要注意的是，每个 Statement 对象都与一个特定的 Connection 对象相关联。你可以使用 Connection 对象的 createStatement() 方法来创建一个新的 Statement 对象。

ResuleSet类

ResultSet 类是Java的JDBC API中的一个接口，它表示数据库查询结果的数据表。你可以使用 ResultSet 对象的方法来遍历查询结果并获取数据。

例如，你可以使用 next() 方法来移动到下一行，使用 getString()、getInt() 等方法来获取当前行中的数据。此外，ResultSet 类还提供了一些方法来更新查询结果中的数据。

需要注意的是，每个 ResultSet 对象都与一个特定的 Statement 对象相关联。当你使用 Statement 对象的 executeQuery() 方法来执行查询操作时，该方法会返回一个新的 ResultSet 对象，用于表示查询结果。

SQL查询语句对数据库的查询操作将返回一个ResultSet对象，ResultSet对象由按“列”组织的数据行构成。

在获取结果集中的结果时，可以使用数据表的字段名也可以使用字段所在列的列数。

注意：无论字段是何种属性，总可以使用getString(int columnIndex)或getString(String columnName)方法返回字段值的串表示。

关闭连接

需要特别注意的是，ResultSet对象和数据库连接对象实现了紧密的绑定，一旦连接对象被关闭，ResultSet对象中的数据立刻消失。这就意味着应用程序在使用ResultSet对象中的数据时必须始终保持和数据库的连接，直到应用程序ResultSet对象中的数据查看完毕。例如，如果在代码

ResultSet rs = sql.executeQuery("select * from students");

之后立刻关闭连接

con.close();

那么程序将无法获取rs中的数据。

控制游标

结果集的游标的初始位置在结果集第一行的前面，结果集调用next()方法向下（后）移动游标，移动成功时返回true，否则返回false。如果需要在结果集中上下（前后）移动、显示结果集中某条记录或随机显示若干条记录，必须返回一个可滚动的结果集。为了得到一个可滚动的结果集，需使用下述方法获得一个Statement对象：

Statement stmt = con.createStatement(int type, int concurrency);

然后根据参数type、concurrency的取值情况，stmt返回相应类型的结果集：

ResultSet rs = stmt.executeQuery(SQL语句);

type取值决定滚动方式，取值如下：

  • ResultSet.TYPE_FORWORD_ONLY：结果集的游标只能向下滚动
  • ResultSet.TYPE_SCROLL_INSENSITIVE：结果集的游标可以上下滚动，当数据库变化时，当前结果集不变
  • ResultSet.TYPE_SCROLL_SENSITIVE：返回可滚动的结果集，当数据库变化时，当前结果集同步改变。
  • ResultSet.CONCUR_READ_ONLY：不能用结果集更新数据库中的表
  • ResultSet.CONCUR_UPDATABLE：能用结果集更新数据库中的表

滚动查询好像经常用到ResultSet的下述方法。

  • previous()：将游标向上移动，该方法返回boolean类型数据，当移动到结果集的第一行之前时返回false
  • beforeFirst()：将游标移到结果集的初始位置，即在第一行之前
  • afterLast()：将游标移动到结果集的最后一行之后
  • first()：将游标移动到结果集的第一行
  • last()：将游标移动到结果集的最后一行
  • isAfterLast()：判断游标是否在最后一行之后
  • isBeforeFirst()判断游标是否在第一行之前
  • isFirst()：判断游标是否指向结果集的第一行
  • isLast()：判断游标是否指向结果集的最后一行
  • getRow()：得到当前游标所指向的行号，行号从1开始，如果结果集没有行，返回0
  • absolute(int row)：将游标移到参数row指定的行

如果row取值为负，就是导数的行数

示例

GetDBConnection.java：

package cn.cukor.one;

import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.SQLException;

public class GetDBConnection {
    public static Connection connectDB(String DBName, String id, String p) {
        Connection connection = null;
        String url = "jdbc:mysql://localhost:3306/" + DBName;
        try {
            Class.forName("com.mysql.cj.jdbc.Driver");
        } catch (ClassNotFoundException e) {
            e.printStackTrace();
        }
        try {
            connection = DriverManager.getConnection(url, id, p);
        } catch (SQLException e) {
            e.printStackTrace();
        }

        return connection;
    }
}

Main.java：

package cn.cukor.one;

import java.sql.Connection;
import java.sql.ResultSet;
import java.sql.SQLException;
import java.sql.Statement;
import java.util.Random;

public class Main {
    public static void main(String[] args) {
        Connection connection;
        Statement sql;
        ResultSet rs;
        connection = GetDBConnection.connectDB("students", "root", "123456");
        if (connection == null) return;
        try {
            sql = connection.createStatement(ResultSet.TYPE_SCROLL_SENSITIVE, ResultSet.CONCUR_READ_ONLY);
            rs = sql.executeQuery("select * from stu");

            rs.last();  // 移动游标到最后一行
            int max = rs.getRow();
            System.out.println("表共有" + max + "条记录，随机抽取两条记录");
            int[] a = {0, 1};
            Random random = new Random();
            a[0] = random.nextInt(2) + 1;   // 游标从1开始计数，如果出现0就会报错
            while (a[0] == a[1]) {
                a[1] = random.nextInt(2);
            }
            System.out.println("a[0] = " + a[0]);
            System.out.println("a[1] = " + a[1]);
            for (int i = 0; i < a.length; i++) {
                rs.absolute(a[i]);     // 游标移动到第i行
                String sno = rs.getString("sno");
                String name = rs.getString("name");
                System.out.println(sno + "\t" + name);
            }

            connection.close();
        } catch (SQLException e) {
            e.printStackTrace();
        }
    }
}
