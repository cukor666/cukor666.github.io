Vue快速入门

介绍

Vue (读音 /vjuː/，类似于 view) 是一套用于构建用户界面的渐进式框架。与其它大型框架不同的是，Vue 被设计为可以自底向上逐层应用。Vue 的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。另一方面，当与现代化的工具链 以及各种支持类库 结合使用时，Vue 也完全能够为复杂的单页应用提供驱动。

Vue是前端框架。可以快速开发前端工程。前后端分离一般使用Vue做前端的部分。

安装

Vue

访问vue2官网：https://v2.cn.vuejs.org/

即可得到vue.js文件，然后就可以在项目中引用vue.js，然后就可以使用Vue这个框架了。

npm

node官网：https://nodejs.org/en

得到一个msi程序，直接运行一路下一步安装即可。

安装完成后可以通过命令行：

npm -v

验证是否安装成功。如果出现版本好，则说明安装成功。

换源：

taobao镜像：npm config set registry https://registry.npm.taobao.org

Vue命令行工具（脚手架）

Vue/Cli官网：https://cli.vuejs.org/zh/index.html

安装vue/cli:

npm install -g @vue/cli

第一个Vue工程

创建一个vue-study文件夹，并使用VSCode打开。

把前端的插件装上。

还有vue的插件。自己搜然后安装即可。

新建一个index.html，并把vue.js通过<script>标签引进来。

index.html：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>index</title>
    <script type="text/javascript" src="js/vue.js" charset="UTF-8"></script>
</head>
<body>
    <div id="app">
        {{ message }}
    </div>
    <script type="text/javascript">
        var app = new Vue({
            el: '#app',
            data: {
                message: 'Hello Vue!'
            }
        });
    </script>
</body>
</html>

运行结果：

我们已经成功创建了第一个 Vue 应用！看起来这跟渲染一个字符串模板非常类似，但是 Vue 在背后做了大量工作。现在数据和 DOM 已经被建立了关联，所有东西都是响应式的。

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    </script>

运行结果：

根据这个特性，我们之后就可以通过后端拿到数据，然后把数据显示在页面上，这样就比较方便。

条件与循环

v-if

<div id="app">
    <p v-if="flag">{{ message }}</p>
</div>
<script type="text/javascript">
    var app = new Vue({
        el: '#app',
        data: {
            message: 'Hello Vue',
            flag: true
        }
    });
</script>

现在flag的值是true，所以网页上可以看得到，但是如果把flag设置为false，那么网页上就看不到message的打印。

v-else-if

<div id="app">
    <p v-if="color === 'red'">red</p>
    <p v-else-if="color === 'blue'">blue</p>
    <p v-else>{{ message }}</p>
</div>
<script type="text/javascript">
    var app = new Vue({
        el: '#app',
        data: {
            message: 'Hello Vue',
            color: 'blue'
        }
    });
</script>

v-else

<div id="app">
    <p v-if="color === 'red'">red</p>
    <p v-else-if="color === 'blue'">blue</p>
    <p v-else>{{ message }}</p>
</div>
<script type="text/javascript">
    var app = new Vue({
        el: '#app',
        data: {
            message: 'Hello Vue',
            color: 'green'
        }
    });
</script>

v-for

<div id="app">
    <ol>
        <li v-for="item in book">
            {{ item.name }}
        </li>
    </ol>
</div>
<script type="text/javascript">
    var app = new Vue({
        el: '#app',
        data: {
            book: [
                { name: 'Java程序设计' },
                { name: 'C语言程序设计' },
                { name: 'C++程序设计' }
            ]
        }
    });
</script>

绑定

v-on（事件绑定）

Vue可以用 v-on 指令添加一个事件监听器，通过它调用在 Vue 实例中定义的方法：

<div id="app">
    <p>{{ message}}</p>
    <button v-on:click="printMessage">按我</button>
</div>
<script type="text/javascript">
    var app = new Vue({
        el: '#app',
        data: {
            message: '再见你好吗'
        },
        methods: {
            printMessage: function () {
                this.message = '不好'
            }
        }
    });
</script>

简写：v-on可以简写成@

<div id="app">
    <p>{{ message}}</p>
    <button @click="printMessage">按我</button>
</div>
<script type="text/javascript">
    var app = new Vue({
        el: '#app',
        data: {
            message: '再见你好吗'
        },
        methods: {
            printMessage: function () {
                this.message = '不好'
            }
        }
    });
</script>

v-model（双向绑定）

Vue 还提供了 v-model 指令，它能轻松实现表单输入和应用状态之间的双向绑定。

<div id="app">
    <p>{{ message}}</p>
    <input v-model="message">
    <button @click="printMessage">按我</button>
</div>
<script type="text/javascript">
    var app = new Vue({
        el: '#app',
        data: {
            message: '再见你好吗'
        },
        methods: {
            printMessage: function () {
                this.message = '不好'
            }
        }
    });
</script>

v-bind（属性绑定）

用于动态改变标签属性中的值。

<div id="app">
    <a v-bind:href="url">{{ urlName }}一下</a>
    <button @click="changeSearch">修改搜索</button>
</div>
<script type="text/javascript">
    var app = new Vue({
        el: '#app',
        data: {
            index: 0,
            linkNames: [
                '百度',
                '京东',
                'Bing'
            ],
            links: [
                'https://www.baidu.com',
                'https://www.jd.com',
                'https://www.bing.com',
            ],
            urlName: '百度',
            url: 'https://www.baidu.com'
        },
        methods: {
            changeSearch: function () {
                this.index = (this.index + 1) % 3
                alert(this.index)
                this.url = this.links[this.index]
                this.urlName = this.linkNames[this.index]
            }
        }
    });
</script>

当按下”修改搜索“的按钮后，url发生改变，然后点击链接可以跳转到对应的网页中。

v-bind可以简写成:

例如上面的：

<a v-bind:href="url">{{ urlName }}一下</a>

可以修改为

<a :href="url">{{ urlName }}一下</a>

效果是一样的。

动态class与style

当某种条件生效的时候才使用特定的样式，否则就不使用。

class

<style>
    .pad {
        padding: 0 10px;
    }

    .active {
        color: red;
    }
</style>
<div id="app" style="display: flex;">
    <select v-model="currentMenu">
        <option value="首页">首页</option>
        <option value="用户列表">用户列表</option>
        <option value="管理员列表">管理员列表</option>
    </select>
    <div class="pad" :class="{ 'active': item == currentMenu }" v-for="item in menus">{{ item }}</div>
</div>
<script type="text/javascript">
    var app = new Vue({
        el: '#app',
        data: {
            menus: [
                '首页',
                '用户列表',
                '管理员列表'
            ],
            currentMenu: '用户列表'
        }
    });
</script>

style

<style>
    .pad {
        padding: 0 10px;
    }

    .active {
        color: red;
    }
</style>
<div id="app" style="display: flex;">
    <select v-model="currentMenu">
        <option value="首页">首页</option>
        <option value="用户列表">用户列表</option>
        <option value="管理员列表">管理员列表</option>
    </select>
    <div class="pad" :style="{ fontSize: item === currentMenu ? '20px' : '15px' }" 
        :class="{ 'active': item === currentMenu }" 
        v-for="item in menus">
        {{ item }}
    </div>
</div>
<script type="text/javascript">
    var app = new Vue({
        el: '#app',
        data: {
            menus: [
                '首页',
                '用户列表',
                '管理员列表'
            ],
            currentMenu: '用户列表'
        }
    });
</script>

Vue脚手架

从这开始之后，前端写vue都不再使用之前的一样，通过HTML的<script>标签把vue.js引进来。而是直接通过npm把vue的工程创建好，在这个工程中去开发前端的代码。

如果还没有安装vue-cli的先安装一下。

npm install -g @vue/cli

然后通过下面的方式检查一下是否安装成功。

vue --version

如果出现版本号，则说明安装成功。

然后进入经常写代码的的文件夹下执行下面的命令创建vue的工程。

vue create 项目名

例如：

vue create vue_project

操作：

  • 上下键切换
  • 空格键选择
  • 回车确认
  • 判断是，大写字母表示默认选项，例如（y/N）默认是N.

默认，直接回车。

默认。直接回车。

然后等待创建成功即可，如果网速太慢，可以先去了解一下怎么换国内的镜像源。

当出现下面的界面的时候，说明已经创建完成了。

cd vue_project 	# 进入vue_project文件夹，也就是我们刚刚创建的工程
npm run serve	# 运行项目，启动服务器服务，然后可以再页面展示vue渲染的页面

出现下面的效果，说明服务器已经运行起来了。ctrl+鼠标右键点击访问Local即可。

运行效果：

这个就是vue的工程的创建，比之前的方式虽然说更麻烦了一定，但是这个比较规范，后续和SpringBoot一起做前后端分离项目的时候，都使用这种方式创建vue工程。

如果想停掉服务器，只需要在命令行按下ctrl+C即可。
