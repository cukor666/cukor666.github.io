Vue循环渲染

在Vue中，v-for指令可以将一个数组中的数据渲染为列表视图。

基本用法

第一种
item in list
第二种
(item, index) in list
第三种
(item, key, index) in list	:key=?			?表示参数，一定是唯一值

上面的格式中，in可以使用of来替换。

  • item：临时变量，遍历list拿到的值
  • list：列表变量本身
  • index：索引，有时候需要所以，可以使用它来接收
  • key：键

示例：

data() {
    return {
        hobby: ['唱', '跳', 'rap', '篮球']
    }
},

第一种：

<div v-for="item in hobby">
    {{item}}
</div>

第二种：

<div v-for="(item, index) in hobby">
    {{index+1}}: {{item}}
</div>

第三种：

<div v-for="(item, key, index) in hobby" :key="index">
    {{key}}: {{item}}
</div>

list也可以是对象，只要是列表的都可以。item是拿到列表中的一个元素。

示例：

<div v-for="(item, key, index) in persons" :key="index">
    {{key}}: {{item}}
</div>
data() {
    return {
        persons: [
            {
                name: '陶喆',
                age: 54
            },
            {
                name: '王力宏',
                age: 50
            },
            {
                name: '方大同',
                age: 40
            }
        ]
    }
},

既然是对象，那就可以通过.的方式访问对象中的属性。

<div v-for="(item, key, index) in persons" :key="index">
    {{key}} -> 姓名：{{item.name}} 年龄：{{item.age}}
</div>

list本身也可以是对象，那么使用v-for去遍历的时候，item会拿到对象中的各个属性。

<div v-for="(item, key, index) in person" :key="index">
   {{key}} : {{item}}
</div>
data() {
    return {
        person: {
            name: '陶喆',
            age: 54,
            addr: '台湾'
        }
    }
},

一般v-for结合列表一起使用的场景比较多。

<ol>
    <li v-for="(item, key, index) in person" :key="index">
        {{key}} : {{item}}
    </li>
</ol>
data() {
    return {
        person: {
            name: '陶喆',
            age: 54,
            addr: '台湾'
        }
    }
},

高级用法

当使用v-for指令对列表进行循环渲染后，实际上就实现了对这个数据对象的绑定，调用下面的函数对列表数据对象进行更新，视图也会对应更新：

函数名说明
push()向列表尾部追加一个元素
pop()删除列表尾部的一个元素
shift()删除列表头部的一个元素
unshift()想列表头部插入一个元素
splice()对列表进行分割操作
sort()对列表进行排序操作
reverse()对列表进行逆序操作
