简单使用Go标准库的os模块。

简介

os，顾名思义就是操作系统，那么它的作用就是对操作系统上的文件、命令、进程等做操作。

os能做的事情如下：

  • 文件和目录的操作
  • 环境变量的访问与获取
  • 执行外部命令
  • 进程管理
  • 文件权限和属性
  • 错误处理
  • 信号处理

创建目录

Mkdir

官方：creates a new directory with the specified name and permission bits (before umask). If there is an error, it will be of type

创建一个带有指定名字和指定权限的新目录，如果有异常就会返回一个错误。

func Mkdir(name string, perm FileMode) error
  • name：要创建的目录名称，亲测可以递归创建目录，但是不能在子目录下创建文件。例如：testdir/subdir/a/b/c没有问题，但是创建完目录后在子目录下面写入文件testdir/subdir/testfile.txt就不行。会报错：mkdir testdir/subdir: The system cannot find the path specified.
  • perm：文件权限，一般在Linux下面对权限有特别的在意，在Windows没有太多在意。

示例：

package main

import (
	"log"
	"os"
)

func main() {
	// 在当前项目目录下面创建一个 testdir 目录（文件夹）
	// 0750 -> 八进制数0开头，它的权限就是750
	// 表示当前用户权限：7	111 -> 可读可写可执行
	// 同组的用户权限：5	101 -> 可读可执行
	// 其他用户权限：0		000 -> 没有权限
	err := os.Mkdir("testdir", 0750)
	// 如果创建目录时出现了错误，并且不是os的错误，
	if err != nil && !os.IsExist(err) {
		log.Fatal(err) // 就直接结束，不再往后执行
	}
	// 如果没有错误
	// 在testdir下面写一个文件，文件名为testfile.txt，写入的内容是Hello, Gophers!
	// 这里写入的内容类型必须是[]byte类型
	// 0660 -> 660 -> 110 110 000
	// -> 当前用户：可读可写，同组用户：可读可写，其他用户：没有权限
	err = os.WriteFile("testdir/testfile.txt", []byte("Hello, Gophers!"), 0660)
	// 如果写入失败
	if err != nil {
		log.Fatal(err) // 直接结束
	}
}

MkdirAll

官方：creates a directory named path, along with any necessary parents, and returns nil, or else returns an error. The permission bits perm (before umask) are used for all directories that MkdirAll creates. If path is already a directory, MkdirAll does nothing and returns nil.

创建一个被命名的目录，可以允许多个父目录，然后返回nil（没有错误），否则返回一个错误。

这个函数就是解决Mkdir的问题，不能在子目录下面创建文件。

// 递归创建文件夹 test/subdir
err := os.MkdirAll("test/subdir", 0750)
if err != nil {
	log.Fatal(err)
}
// 在子目录下创建testfile.txt文件，并写入内容 Hello, Gophers!
err = os.WriteFile("test/subdir/testfile.txt", []byte("Hello, Gophers!"), 0660)
if err != nil {
	log.Fatal(err)
}

MkdirTemp

官方：creates a new temporary directory in the directory dir and returns the pathname of the new directory. The new directory’s name is generated by adding a random string to the end of pattern. If pattern includes a “*“, the random string replaces the last “*“ instead. If dir is the empty string, MkdirTemp uses the default directory for temporary files, as returned by TempDir. Multiple programs or goroutines calling MkdirTemp simultaneously will not choose the same directory. It is the caller’s responsibility to remove the directory when it is no longer needed.

在dir目录下创建一个新的临时目录，然后返回这个新目录的路径名称。

这个新的目录名是通过在模式末尾添加一个随机字符串生成的。

如果模式中包含“*”，则随机字符串将替换最后一个“*”。

如果dir是空字符串，MkdirTemp将使用TempDir作为默认。

同时调用MkdirTemp的多个程序或goroutine不会选择同一目录。当不再需要该目录时，调用者有责任删除该目录。

// 创建一个临时文件夹，使用默认的名称
dir, err := os.MkdirTemp("", "example")
if err != nil {
	log.Fatal(err) // 创建失败就直接结束
}
// 最后在return直接把临时文件夹进行删除
defer os.RemoveAll(dir) // clean up

// 加入一个临时文件
file := filepath.Join(dir, "tmpfile")
// 往这个文件写点内容
if err := os.WriteFile(file, []byte("content"), 0666); err != nil {
	log.Fatal(err) // 如果写入失败就直接结束
}

删除目录或文件

Remove

func Remove(name string) error

示例：

// 删除 testdata 目录下面的 hello 文件
err := os.Remove("testdata/hello")
if err != nil {
	log.Fatal(err) // 如果删除失败就直接结束
}

上面的例子是删除文件

// 删除 testdata 目录
err := os.Remove("testdata")
if err != nil {
	log.Fatal(err) // 如果删除失败就直接结束
}

RemoveAll

删除所有，包括子目录、子目录下的文件。

func RemoveAll(path string) error

示例：

// 事先创建好多级目录和文件

// 删除a目录。包括a目录下的所有东西
err := os.RemoveAll("a")
if err != nil {
	log.Fatal(err)
}

重命名

func RemoveAll(path string) error
// 将test1目录重命名为test2
err := os.Rename("test2", "test3")
// 如果test3已经存在，并且是一个文件，那么test2就重命名为test3，并且原来的test3文件就被替换（删除）
if err != nil {
	log.Fatal(err) // 出错就直接返回
	// 如果test3已经存在，并且是一个目录，就会报下面的错
	// rename test2 test3: Access is denied.
}

获取当前工作目录

返回与当前目录相对应的根路径名。如果可以通过多个路径（由于符号链接）访问当前目录，Getwd可能会返回其中任何一个路径。

func Getwd() (dir string, err error)

示例：

dir, err := os.Getwd()
if err != nil {
	log.Fatal(err)
}
log.Println(dir)
D:\myCoding\Go\go-terminate

修改目录

将当前工作目录更改为指定目录。如果有一个错误，它的类型将是*PathError。

func Chdir(dir string) error

示例：

dir, err := os.Getwd()
if err != nil {
	log.Fatal(err)
}

log.Printf("原目录dir: %v\n", dir)
err = os.Chdir("d:/")
if err != nil {
	log.Fatal(err)
}

dir, err = os.Getwd()
if err != nil {
	log.Fatal(err)
}
log.Printf("新目录dir: %v\n", dir)
2023/10/23 14:22:15 原目录dir: D:\myCoding\Go\go-terminate
2023/10/23 14:22:15 新目录dir: d:\

查看目录或文件状态

可以使用os模块的Stat函数来检查文件或目录是否存在。如果文件或目录存在，则Stat函数不会返回错误。

func Stat(name string) (fs.FileInfo, error)
_, err := os.Stat("test3")
if os.IsNotExist(err) {
	fmt.Println("文件或目录不存在")
	return
}
fmt.Println("文件或目录存在")

创建文件

Create

func Create(name string) (*File, error)
// 在当面目录下创建一个名为 hello.go 的文件
f, err := os.Create("hello.go")
if err != nil {
	log.Fatal(err) // 创建失败就直接结束
}
// 如果创建成功，就往文件中添加点内容
if _, err := f.WriteString("package hello"); err != nil {
	log.Fatal(err) // 添加失败就直接结束
}
// 记得关闭
if err := f.Close(); err != nil {
	log.Fatal(err)
}

打开关闭文件

file.txt：

月亮代表谁的心

Open

func Open(name string) (*File, error)
// 打开file.txt文件
f, err := os.Open("file.txt")
if err != nil {
	// 如果文件不存在就打印错误信息
	fmt.Printf("err: %v\n", err)
}
defer f.Close()          // 记得关闭文件
s := f.Name()            // 如果没有报错，说明文件存在，那就获取它的名字
fmt.Printf("s: %v\n", s) // 打印文件名

如果打开一个不存在的文件就会报错：

err: open file2.txt: The system cannot find the file specified.

OpenFile

func OpenFile(name string, flag int, perm FileMode) (*File, error)
// 打开file2.txt文件
// O_RDWR -> 只读只写
// O_CREATE -> 只创建
// 连起来就是有这个文件就打开，然后只能读和写，如果没有就创建出来，然后读写
// 660 -> 110 110 000 -> 可读可写 可读可写 无权限
f, err := os.OpenFile("file2.txt", os.O_RDWR|os.O_CREATE, 0660)
if err != nil {
	fmt.Printf("err: %v\n", err)
	return
}
defer f.Close()
fmt.Printf("f.Name(): %v\n", f.Name())

读取文件

ReadFile

// 运行之前一定要确保有testdata目录，并且下面有一个hello文件，没有后缀名

// 读取testdata目录下面的hello文件。
data, err := os.ReadFile("testdata/hello")

if err != nil {
	log.Fatal(err) // 如果读取失败就直接结束
}

// 读取成功就将hello中的内容打印到标准输出流（控制台终端）上
os.Stdout.Write(data)
