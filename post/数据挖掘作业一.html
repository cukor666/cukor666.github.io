数据挖掘作业一（第一章）

一、多项选择题

下列各题中有一个或多个正确选项。

1、下列任务中，属于数据挖掘技术在商务智能方面应用的是( )

A、顾客分析 B、定向营销 C、商店分布 D、欺诈检测

答案：ABCD

2、下列任务中，属于数据挖掘范畴的是( )

A、预测一位新的顾客是否会在一家百货公司消费 100 美元以上；

B、使用数据库管理系统查找个别的记录；

C、通过因特网的搜索引擎查找特定的 Web 页面；

D、根据性别划分公司的顾客。

答案：A

3、下列任务中，属于数据挖掘范畴的是( )

A、根据可赢利性划分公司的顾客；

B、计算公司的总销售额；

C、按学生的标识号对学生数据库排序；

D、监视病人心率的异常变化。

答案：AD

4、数据预处理涉及的步骤包括( )

A、融合来自多个数据源的数据；

B、清洗数据以消除噪声和重复的观测值；

C、选择数据挖掘的方法和技术；

D、选择与当前数据挖掘任务相关的记录和特征。

答案：ABD

C选项属于数据挖掘的阶段了，不属于预处理阶段。

5、下列数据挖掘任务中，属于分类任务的是( )

A、预测一个 Web 用户是否会在网上书店买书；

B、预测某股票的未来价格；

C、根据检查结果判断病人是否患有某种疾病；

D、网络的入侵检测。

答案：ACD

B选项属于回归任务。

6、异常检测的应用包括( )

A、检测欺诈 B、网络攻击 C、疾病的不寻常模式 D、生态系统扰动

答案：ABCD

二、名词解释

1、数据挖掘

数据挖掘是在大型数据库中，自动地发现有用信息的过程。

2、数据预处理

数据预处理是将未加工的输入数据转换成适合分析的形式。

3、关联分析

关联分析用来发现描述数据中强关联特征的模式。

4、聚类分析

聚类分析用来发现紧密相关的观测值组群，使得与属于不同簇的观测值相比，属于同一簇的观测值相互之间尽可能类似。

5、异常检测

异常检测的任务是识别其特征显著不同于其他数据的观测值。

2、数据挖掘算法要解决的问题包括哪些？

（1）算法的可伸缩性 （2）高维性 （3）异种数据和复杂数据 （4）数据的所有权与分布 （5）非传统的分析

3、数据挖掘任务分为哪两类？

数据挖掘任务分为下面两大类：

（1）预测任务。其目标是根据其他属性的值，预测特定属性的值。

（2）描述任务。其目标是导出概括数据中潜在联系的模式(相关、趋势、聚类、轨迹和 异常)。

4、预测建模任务有哪两类？它们的区别是什么？

预测建模任务分为分类和回归两类。

它们的区别在于分类用于预测离散的目标变量;而回归用于预测连续的目标变量
