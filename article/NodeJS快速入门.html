NodeJS快速入门

简介

node.js只是JavaScript的运行环境，所以学习node.js其实就是学JavaScript，但是此JavaScript不是浏览器中的JavaScript。浏览器的JavaScript可以操作DOM和BOM不能操作磁盘文件；Node.js中的JavaScript不能操作DOM和BOM但可以操作磁盘文件。

基本语法

变量、常量、表达式、分支、循环还有函数等基础的JS语法Node中和浏览器中都是一样的。

全局对象

__filename

__filename 表示当前正在执行的脚本的文件名。它将输出文件所在位置的绝对路径，且和命令行参数所指定的文件名不一定相同。 如果在模块中，返回的值是模块文件的路径。

__dirname

__dirname 表示当前执行脚本所在的目录。

文件系统

这个是区别于浏览器JavaScript的，导入fs模块即可操作磁盘中的文件。

读取操作

新建一个example.txt文件：

然后在主js文件中写入以下代码来读取example.txt中的文件：

xxxxxxxxxx
const fs = require("fs");


fs.readFile("example.txt", "utf8", (err, data) => {
  if (err) {
    console.error(err);
    return;
  }
  console.log("文件的内容为:", data);
});

  • 第一个参数是要读取的文件的路径

  • 第二个参数是读取文件的格式，一般默认为utf8，可以写

  • 第三个参数是回调函数

    • 第一个参数是读取失败时获取到的错误对象
    • 第二个参数是读取成功时取到的文件中的内容

GET请求

需要解析url路径，所以需要引入url模块：

x
const url = require('url');

写一个案例

xxxxxxxxxx
const http = require("http");
const url = require("url");


const server = http.createServer((req, res) => {
  // 解析请求路径
  const parsedUrl = url.parse(req.url, true);
  // 获取请求路径
  const pathname = parsedUrl.pathname;
  // 获取请求方法
  const method = req.method;


  res.setHeader("Content-Type", "text/html");


  if (method === "GET" && pathname === "/") {
    res.statusCode = 200;
    res.end("<h1>Welcome to the Homepage!</h1>");
  } else if (method === "GET" && pathname === "/about") {
    res.statusCode = 200;
    res.end("<h1>About Us</h1>");
  } else {
    res.statusCode = 404;
    res.end("<h1>Page Not Found</h1>");
  }
});


const port = 3000;
server.listen(port, () => {
  console.log(`Server is running on http://localhost:${port}/`);
});

返回JSON数据

需要设置头部参数

x
res.setHeader('Content-Type', 'application/json');

整体代码：

xxxxxxxxxx
const http = require("http");
const url = require('url');


const server = http.createServer((req, res) => {
  const method = req.method; // 获取请求方法
  const url = req.url; // 获取请求的URL


  if (method === "GET" && url === "/data") {
    const data = {
      id: 1,
      name: "John Doe",
      email: "john@example.com",
    };


    res.statusCode = 200;
    res.setHeader("Content-Type", "application/json");
    res.end(JSON.stringify(data));
  } else {
    res.statusCode = 404;
    res.setHeader("Content-Type", "application/json");
    res.end(JSON.stringify({ message: "Resource not found" }));
  }
});


const port = 3000;
server.listen(port, () => {
  console.log(`服务器运行在 http://localhost:${port}/`);
});

统一返回类型

统一返回以下格式的JSON：

xxxxxxxxxx
{
    "code": 200,
    "msg": "消息",
    "data": "具体数据"
}
  • code：状态码
  • msg：消息
  • data：具体数据，可以是字符串，也可以是一个对象，也可以是数组等。

POST请求

xxxxxxxxxx
const http = require('http');


const server = http.createServer((req, res) => {
  if (req.method === 'POST' && req.url === '/login') {
    let body = '';


    // 监听请求体数据
    req.on('data', chunk => {
      body += chunk.toString();
    });


    // 请求体数据接收完毕
    req.on('end', () => {
      try {
        const user = JSON.parse(body);
        console.log(user);


        const response = {
          code: 200,
          msg: "请求成功",
          data: user
        };


        res.writeHead(200, {'Content-Type': 'application/json'});
        res.end(JSON.stringify(response));
      } catch (e) {
        res.writeHead(400, {'Content-Type': 'application/json'});
        res.end(JSON.stringify({code: 400, msg: "Invalid JSON", data: {}}));
      }
    });
  } else {
    res.writeHead(404, {'Content-Type': 'application/json'});
    res.end(JSON.stringify({code: 404, msg: "Not Found", data: {}}));
  }
});


const port = 3000;
server.listen(port, () => {
  console.log(`Server is running on http://localhost:${port}/`);
});
